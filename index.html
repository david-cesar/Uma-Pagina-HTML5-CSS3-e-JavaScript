HTML 5

A Linguagem de Marcação de Hiper Texto - HTML, é a linguagem usada para montar a estrutura de uma página, podemos dizer que ela é o esqueleto que sustenta e dá sentido ao projeto. É com HTML que vamos organizar os textos, imagens, formulários e demais elementos necessários na página. Como os navegadores não sabem diferenciar um título ou um parágrafo, precisamos indicar isso explicitamente em nosso código.

Fazemos isso colocando os elementos estruturais da página entre "tags", que são informações que ficam entre os símbolos de menor < e maior >
